#hubot-darla

A hubot script for Kodi, NZBGet, Sonarr, CouchPotato

## Installation

Add hubot-darla to your `package.json` file:

```json
"dependencies": {
  "hubot": ">= 2.5.1",
  "hubot-scripts": ">= 2.4.2",
  "hubot-darla": ">= 0.0.0"
}
```

Add hubot-darla to your `external-scripts.json`:

```json
["hubot-darla"]
```

Run `npm install hubot-darla`
